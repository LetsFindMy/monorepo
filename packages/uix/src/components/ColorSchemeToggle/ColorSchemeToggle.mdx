# ColorSchemeToggle

A component that allows users to toggle between light, dark, and automatic color schemes.

## Features

- Three-button interface for theme selection
- Uses Mantine's color scheme management
- Responsive layout with proper spacing
- Supports light/dark theme visual testing

## Usage

```tsx
import { ColorSchemeToggle } from './ColorSchemeToggle';

function Demo() {
  return <ColorSchemeToggle />;
}
```
